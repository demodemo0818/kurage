# Kurage v0.12.1-beta

対象バージョン: **0.12.1-beta**
配布物: Android APK / Windows zip（ポータブル・未署名）/ Web（https://kurage.demo2.jp）

> ベータ版です。原則データは保持されますが、やむを得ない場合は事前告知します。
> Windows zip は未署名のため、初回は「Windows によって PC が保護されました」→「詳細情報」→「実行」で起動してください。

今回はバグ修正が中心の小さな更新です（PATCH 上がり = 挙動を大きく変える変更はありません）。

## 🛠 改善・修正

- **引用** — 引用元の URL がまれに正しく解決されない不具合を修正しました。あわせて、引用元の投稿を「投稿元サーバで開く」メニューを追加しました。
- **タイムライン** — 表示名に含まれる双方向制御文字（文字の並びを左右反転させる特殊文字）が、隣の `@ID` や数字の並びに漏れて表示が崩れる問題を修正しました。名前自体の見た目はこれまで通り保持します。
- **プロフィール** — Mastodon 4.6 未満のサーバーで、4.6 専用の UI が誤って表示されることがあったのを修正し、非対応サーバーでは隠すようにしました。

## 📝 既知の注意点

- アルファ／ベータ共通として、サーバー側の仕様差により一部表示が崩れる場合があります。気づいた点があればお知らせください。
