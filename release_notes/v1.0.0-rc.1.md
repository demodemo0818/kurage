# 🪼 Kurage v1.0.0-rc.1 リリースノート

**正式版 v1.0.0 に向けた最初のリリース候補 (Release Candidate) です。** プッシュ通知の見た目を大幅に強化したほか、タイムライン・通知・ギャラリーの細かな不具合を修正しました。このバージョンで問題がなければ、次が正式版 v1.0.0 になります。

- 配布: Android (Google Play クローズドテスト) / Windows ポータブル版 (zip) / Web https://kurage.demo2.jp
- リリース候補版です。不具合を見つけたら、ぜひこのタイミングでお知らせください 🙏
- Windows 版は未署名のため、初回に「Windows によって PC が保護されました」と出たら「詳細情報」→「実行」で起動できます（同梱の「お読みください.txt」にも記載）。

## ✨ 新機能

- **プッシュ通知が誰からのどんな通知か一目でわかるようになりました (Android)**
  - 通知に**リアクションした人のアバター**が丸く表示されます。
  - タイトル先頭に**種別の絵文字**が付きます: ⭐ お気に入り / 🔁 ブースト / 💬 メンション・返信 / 🗨️ 引用 / 😀 絵文字リアクション / 👤 フォロー / 📊 アンケート など。
  - 通知の**種別ごとにチャンネルが分かれました**。Android の「設定 → アプリ → Kurage → 通知」から「お気に入りは無音にしてメンションだけ鳴らす」のような細かい制御ができます。

## 🛠 改善・修正

- **展開した「もっと見る」が勝手に閉じる問題を修正しました**
  - 折りたたみ行数を設定していると、タイムラインに新着が流れてくるたび・スクロールして戻ってくるたびに、展開した投稿がまた折りたたまれてしまう問題を修正しました。
- **通知ページでお気に入り・ブーストのマークが消える問題を修正しました**
  - 通知ページでリアクションした直後に、通知リストの更新でマークの点灯が消えたり別の投稿に付いて見えたりする問題を修正しました。
- **「C#」などの # をハッシュタグと誤認識する問題を修正しました**
  - 「C#が好き」のように単語の途中にある # を、本来ハッシュタグではないのにリンク化してしまう問題を修正しました。メールアドレスの @ の誤認識も同様に修正しています。
- **Web 版: 画面幅を狭くしたとき、画像ビューアで画像送りができない問題を修正しました**
  - マウスのドラッグでスワイプできるようにしたほか、左右の送りボタンと矢印キー (←→、Esc で閉じる) にも対応しました (Web / Windows 版)。

## 📝 既知の注意点

- 通知チャンネルの分割に伴い、以前のバージョンで通知音などをカスタマイズしていた場合は、新しいチャンネルに対して**再設定が必要**です（初回は全チャンネルがデフォルト設定で作られます）。
