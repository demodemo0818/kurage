# 🪼 Kurage v1.0.0

Flutter 製の Mastodon クライアント **Kurage** の最初の公開リリースです。

Kurage は複数アカウント・マルチカラムでの閲覧を軸にした、日本語 UI の非公式 Mastodon クライアントです (英語 UI は今後対応予定)。このリリースからソースコードを Apache-2.0 で公開しています。

## 主な機能

- **マルチアカウント / マルチカラム** — 複数アカウントのタイムラインを 1 カラムにまとめる「統合タイムライン」にも対応
- **ストリーミング (SSE)** — 新着の即時反映、切断時の自動再接続・取り逃し復旧つき
- **プッシュ通知** (Android) / 通知種別ごとのチャンネル分け
- **引用表示** — Mastodon 4.4 公式形式に対応 (Misskey / Fedibird 互換フォールバックあり)
- 予約投稿・下書き・複数アカウントへのクロスポスト、カスタム絵文字・リアクション
- PIN / 生体認証によるアプリロック、設定・アカウントのバックアップ / 復元

## 入手方法

| プラットフォーム | 入手先 |
|---|---|
| Android | [Google Play](https://play.google.com/store/apps/details?id=jp.demo2.kurage) |
| Web | https://kurage.demo2.jp (インストール不要) |
| Windows | このページ下の **Assets** からダウンロード |

### Windows 版について

- **インストーラー版**: `kurage-v1.0.0-setup.exe` — 実行するとユーザー単位でインストールされ、スタートメニューに登録されます (管理者権限不要)。
- **ポータブル版**: `kurage-v1.0.0-windows.zip` — 解凍して `kurage.exe` を実行するだけ。
- どちらも未署名のため、初回に「Windows によって PC が保護されました」と表示されたら「**詳細情報**」→「**実行**」で起動できます。
