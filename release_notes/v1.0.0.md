# 🪼 Kurage v1.0.0 リリースノート

**Kurage 正式版 v1.0.0 のリリースです！** 🎉

昨年からのアルファ・ベータ・リリース候補を経て、ついに正式版になりました。テストにご協力いただいた皆さま、本当にありがとうございました。

- 配布: Android (Google Play) / Windows ポータブル版 (zip) / Web https://kurage.demo2.jp
- Windows 版は未署名のため、初回に「Windows によって PC が保護されました」と出たら「詳細情報」→「実行」で起動できます（同梱の「お読みください.txt」にも記載）。

## ✨ このバージョンについて

機能面はリリース候補 v1.0.0-rc.1 と同じです。rc.1 で大きな問題の報告がなかったため、正式版として公開します。rc.1 からの変更は以下のとおりです。

- **オープンソース化の準備を行いました**
  - ライセンス (LICENSE)・README・コントリビューションガイドを追加し、ソースコードを公開できる状態に整理しました。アプリの動作に変更はありません。

rc.1 の新機能・修正内容（プッシュ通知のアバター表示・種別絵文字・種別別チャンネル、タイムライン・通知・ギャラリーの不具合修正など）は [v1.0.0-rc.1 のリリースノート](v1.0.0-rc.1.md) をご覧ください。

## 📝 既知の注意点

- rc.1 以前のバージョンから更新する場合、通知チャンネルの分割に伴い、以前カスタマイズしていた通知音などは新しいチャンネルに対して**再設定が必要**です。
