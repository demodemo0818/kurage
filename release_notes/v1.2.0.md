# 🪼 Kurage v1.2.0

**Windows 版で動画再生に対応**したほか、クラッシュレポートに基づく**安定性の改善**を多数行いました。

## ✨ 新機能・改善

- **Windows: 投稿内の動画を再生できるようになりました** — これまで Windows 版ではサムネイル表示のみだった動画添付を、アプリ内でそのまま再生できます
- **Misskey 由来の GIF (gifv) が再生されない問題を修正** — Misskey サーバーの投稿に添付されたアニメーション GIF が正しく動くようになりました

## 🛠 安定性の改善 (クラッシュ修正)

Firebase Crashlytics のレポートを分析し、報告数の多い順に以下のクラッシュを修正しました。

- 通知カラムの表示切替時にまれに落ちる問題 (報告数最多)
- タイムラインの追加読み込み (下スクロール) 時にまれに落ちる問題
- プロフィール画面のページング中に画面を閉じると落ちる問題
- 投稿画面でメディアピッカーを開いている間に画面が破棄されると落ちる問題
- 投稿画面で本文にフォーカスせず「#」ボタンを押すと落ちる問題
- 破損画像の読み込み失敗が「クラッシュ」として誤集計されていた問題

## 🐛 その他の修正

- Web: 日付入力欄に極端に大きい数字を入れると入力欄が壊れる問題を修正

| Android | [Google Play](https://play.google.com/store/apps/details?id=jp.demo2.kurage) |

| Windows | このページ下の **Assets** からダウンロード |

### Windows 版について

- **インストーラー版**: `kurage-v1.2.0-setup.exe` — 実行するとユーザー単位でインストールされ、スタートメニューに登録されます (管理者権限不要)。
- **ポータブル版**: `kurage-v1.2.0-windows.zip` — 解凍して `kurage.exe` を実行するだけ。
- どちらも未署名のため、初回に「Windows によって PC が保護されました」と表示されたら「**詳細情報**」→「**実行**」で起動できます。

## English

**Video playback is now supported on Windows**, and animated GIFs from Misskey servers (gifv) play correctly. This release also includes many **stability improvements**: based on crash report analysis, we fixed the most frequent crashes in the notifications column, timeline infinite scroll, profile paging, the media picker, and hashtag insertion, and corrected image-loading noise that was miscounted as crashes.
