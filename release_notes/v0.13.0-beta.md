# Kurage v0.13.0-beta

対象バージョン: **0.13.0-beta**
配布物: Android（Google Play クローズドテスト）/ Android APK / Windows zip（ポータブル・未署名）/ Web（https://kurage.demo2.jp）

> ベータ版です。原則データは保持されますが、やむを得ない場合は事前告知します。
> Windows zip は未署名のため、初回は「Windows によって PC が保護されました」→「詳細情報」→「実行」で起動してください。

今回から **Google Play のクローズドテスト** での配布を開始します。

## ✨ 新機能

- **プッシュ通知のオン/オフ設定** — 設定からプッシュ通知の受信を切り替えられるようになりました。オフにするとサーバー側の購読も解除します。ログアウト時にも自動で購読解除するようにしました。

## 🛠 改善・修正

- **プライバシー** — 広告 ID（AAID）を収集しない設定にしました。

## 📝 既知の注意点

- アルファ／ベータ共通として、サーバー側の仕様差により一部表示が崩れる場合があります。気づいた点があればお知らせください。
