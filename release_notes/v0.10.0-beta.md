# 🪼 Kurage v0.10.0-beta リリースノート

**ここから「ベータ版」になりました。** 主要な機能とデータ形式が一通り固まり、テスターを広げて大丈夫な品質に達したと判断したための節目です（v0.9.x までは「アルファ版」）。前回（v0.9.0-alpha、Windows は v0.9.1-alpha）からの更新です。

- Android APK / Windows ポータブル版(https://drive.google.com/drive/folders/11BcgRYovEwszU9YxznooIFreTAllvtQK?usp=drive_link)
- Web 版（https://kurage.demo2.jp）
- **ベータ移行にあたり、原則としてアプリのデータ（アカウント・カラム・設定）は今後も保持されます。** やむを得ずデータ移行が必要なときは事前にお知らせします（これまでの「まれに再ログイン・データ消去が起こる」運用から変わりました）。
- Windows 版は未署名のため、初回に「Windows によって PC が保護されました」と出たら「詳細情報」→「実行」で起動できます（同梱の「お読みください.txt」にも記載）。

## ✨ 新機能・改善

- **ミュート / ブロックの一覧・管理画面を追加**
  - ミュートした人・ブロックした人・ブロックしたドメインを一覧で確認し、その場で解除できるようになりました。
- **リモートアカウントを相手サーバー / ブラウザで開けるように**
  - 他サーバーのアカウントを、ブラウザや相手サーバー上で直接開く導線を追加しました。
- **他サーバー（Misskey 含む）の読み込みを改善**
  - Misskey 系サーバーのプロフィール等を開いたときに、うまく扱えない場合のグレースフルな案内を追加しました。あわせて関連の不具合も修正しています。
- **Windows / Linux でアプリから直接ログインできるように**（Windows 版は前回 v0.9.1 で先行提供済み）
  - 「アカウントを追加」→ インスタンス URL 入力 →「認証開始」で既定ブラウザが開き、認可するとアプリに戻ってログインが完了します。

## 🛠 改善・修正

- **タイムラインのギャップ（抜け）まわりの位置ずれを修正**
  - ギャップを「下側をキープ」で読み込んだときに、表示位置が大きく上にずれてしまう問題を修正しました。
  - ストリーミング受信中にギャップ表示が消えてしまう問題、未読数バッジの数が合わなくなる問題も修正しています。
- **通知の不具合を修正**
  - アカウントの選択をすべて解除しているのに、操作のタイミングによって通知が表示されてしまうことがある問題を修正しました。
- **ブースト経由で流れてきた投稿のリアクション状態が反映されない問題を修正**
- **アプリロックの不具合を修正**
  - タイムアウトを指定しているとき、バックグラウンドから復帰してもロック画面が出ないことがある問題を修正しました（設定 → アプリロック）。
- **検索でのアカウント表示を改善**（表示名と @ID を横並び一行で表示）
- **特殊なサーバー構成（WEB_DOMAIN ≠ LOCAL_DOMAIN）でプロフィールが正しく開けるように**
- クラッシュ・セキュリティ・堅牢性に関わる複数の問題を、内部監査をもとに一括で修正しました。

## 🔧 内部改善（動作には影響しません）

- 依存パッケージのメジャー更新を含む全面的なアップデートと、自動テストの大幅な拡充を行い、安定性を底上げしました。
