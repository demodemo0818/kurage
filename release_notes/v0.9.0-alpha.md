# 🪼 Kurage v0.9.0-alpha リリースノート

前回（v0.8.2-alpha）からの更新です。デスクトップまわりの新機能と、通知まわりの不具合修正が中心です。

- Android APK / Windows ポータブル版(https://drive.google.com/drive/folders/11BcgRYovEwszU9YxznooIFreTAllvtQK?usp=drive_link)
- Web 版（https://kurage.demo2.jp）
- これはクローズドアルファ版です。仕様変更や、まれに再ログイン・データ消去が起こる場合があります。
- Windows 版は未署名のため、初回に「Windows によって PC が保護されました」と出たら「詳細情報」→「実行」で起動できます（同梱の「お読みください.txt」にも記載）。

## ✨ 新機能・改善

- **デスクトップで画像の保存先を選べるように**（Windows / macOS / Linux）
  - 画像を保存するときに保存先フォルダ・ファイル名を指定できるようになりました。
  - 保存が完了すると通知が前面に表示され、保存できたことが分かりやすくなりました。
- **Windows デスクトップ版が登場**（ポータブル zip）
  - 解凍して `kurage.exe` を実行するだけ。インストール不要です（VC++ ランタイム同梱なので別途インストール不要）。
  - アプリ名・アイコン・製品情報を「Kurage」ブランドに整備しました。

## 🛠 修正

- **引用投稿の通知がリロードで消える不具合を修正**
  - 引用された通知が、通知一覧を読み込み直すと消えてしまうことがありました。読み込み直しても残るようになりました。
- **効果音設定の説明文を修正**
  - 「.mp3 が必要」という誤解を招く文言を削除しました（設定 → 効果音）。
