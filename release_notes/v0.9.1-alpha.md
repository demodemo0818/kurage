# 🪼 Kurage v0.9.1-alpha リリースノート

**Windows 版のみの更新です。** Android / Web は v0.9.0-alpha のままで変更ありません。

- 配布: Windows ポータブル版（zip）のみ
- これはクローズドアルファ版です。仕様変更や、まれに再ログイン・データ消去が起こる場合があります。
- Windows 版は未署名のため、初回に「Windows によって PC が保護されました」と出たら「詳細情報」→「実行」で起動できます（同梱の「お読みください.txt」にも記載）。

## 🛠 修正

- **Windows で Mastodon に直接ログインできるようになりました**
  - これまで Windows 版はアプリ内から直接ログインできず、他端末からのバックアップ復元頼みでした。
  - 「アカウントを追加」→ インスタンス URL 入力 →「認証開始」で**既定ブラウザが開き**、認可するとアプリに戻ってログインが完了します。
- **Windows から投稿したときのアプリ名（via）が正しくなりました**
  - 以前は復元元（Web 等）のアプリ名のまま「Kurage for mastodon (Web)」と表示されていました。
  - **Windows で新しくログインし直したアカウント**なら、投稿の via が「Kurage for mastodon (Windows)」になります。

## 📝 既知の注意点

- **これまでバックアップ復元で入れていたアカウントは、via が「(Web)」のまま**です。これはアプリ名がアクセストークンにサーバ側で固定されているためで、**Windows で新規ログインし直したアカウント**でのみ正しい表示になります（必要なら古いアカウントは削除してください）。
- 過去の投稿の via は履歴のため変わりません。新規ログイン後の投稿から正しくなります。
