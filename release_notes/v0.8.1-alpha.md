# 🪼 Kurage v0.8.1-alpha リリースノート

前回（v0.7.0-alpha）からの更新です。0.8.0 は配信せず、まとめて **0.8.1** としてお届けします。

- 配布: Android APK / Web 版（https://kurage.demo2.jp）
- これはクローズドアルファ版です。仕様変更や、まれに再ログイン・データ消去が起こる場合があります。

## ✨ 新機能

- **効果音に対応しました** 🔔
  「通知の受信」「投稿の完了」「タイムラインの引っ張って更新」のタイミングで音を鳴らせます。
  - **初期状態はオフ**です。**設定 → アプリ設定 → サウンド（効果音）** から、鳴らしたいものを個別にオンにしてください。
  - 音はアプリを開いている間（フォアグラウンド）のみ鳴ります。
  - 効果音素材は [OtoLogic](https://otologic.jp)（CC BY 4.0）を使用しています。

- **ボスキー（偽装モード）を試験追加**（Web / デスクトップ限定）🫣
  キーボードの **F9** でアプリの見た目を某検索サイト風に切り替えられる隠し機能です。設定でオンにすると使えます。アカウント切り替え・タイムライン切替タブ・長文の展開にも対応。

- **画像を手動でぼかす/隠すボタンを追加**
  NSFW 指定でない画像でも、自分でぼかし・非表示にできるようになりました。

- **ライセンス・クレジット画面を追加**
  設定 → アプリ情報 →「ライセンス・クレジット」で、使用素材とオープンソースライセンスを確認できます。

- **（Deck/ワイド表示）画像ビューアを最適化**
  画像をアスペクト比に合わせた窓で表示し、デスクトップ操作にも対応しました。

## 🛠 改善・修正

- 投稿に失敗したとき、入力した内容が消えないようにしました。
- Web 版の不具合を修正：「URLを共有」が動かない／「Google翻訳で開く」が動かない／iOS Safari でアカウント追加の認証画面が出ない。
- アカウントのハンドル表示から `https://` が付いてしまう表示を修正。
- （Deck）狭いカラムでグループ通知の「全員を見る」が消えてしまう問題を修正。
- （Deck）サーバー情報ダイアログの幅を制限して見やすく。
- （Deck）投稿ペインからの下書き・予約投稿管理をポップアップ表示に統一。

## 📝 既知の注意点

- Web 版の効果音は、ブラウザの仕様で**最初に画面を 1 回操作するまで鳴らない**ことがあります（操作後は鳴ります）。
- ボスキーは Web / デスクトップのみの機能です（スマホ版には出ません）。
