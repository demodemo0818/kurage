# 🪼 Kurage v1.1.0

**英語 UI (English UI) に対応しました。** 設定 → 外観設定 → 表示言語 から「システム / 日本語 / English」を切り替えられます (既定はシステム言語に追従。日本語・英語以外の言語では英語表示にフォールバックします)。

## 主な変更

- **英語UI対応** — アプリ内のほぼ全ての文言を英語に翻訳し、設定画面に言語ピッカーを追加
- README に英語版 ([README.en.md](https://github.com/demodemo0818/kurage/blob/main/README.en.md)) を追加
- Web 版の OAuth ログイン確認画面も日英対応
- 紹介サイト (https://demo2.jp/kurage/) も日英切り替えに対応

## 入手方法

| プラットフォーム | 入手先 |
|---|---|
| Android | [Google Play](https://play.google.com/store/apps/details?id=jp.demo2.kurage) |
| Web | https://kurage.demo2.jp (インストール不要) |
| Windows | このページ下の **Assets** からダウンロード |

### Windows 版について

- **インストーラー版**: `kurage-v1.1.0-setup.exe` — 実行するとユーザー単位でインストールされ、スタートメニューに登録されます (管理者権限不要)。
- **ポータブル版**: `kurage-v1.1.0-windows.zip` — 解凍して `kurage.exe` を実行するだけ。
- どちらも未署名のため、初回に「Windows によって PC が保護されました」と表示されたら「**詳細情報**」→「**実行**」で起動できます。

## English

Kurage now supports an **English UI**, alongside Japanese. You can switch languages under Settings → Appearance → Display language ("Follow system" / "日本語" / "English"). The default follows your device's language, and falls back to English for unsupported languages.

See [README.en.md](https://github.com/demodemo0818/kurage/blob/main/README.en.md) for the English version of the project README.
