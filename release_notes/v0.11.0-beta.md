# Kurage v0.11.0-beta リリースノート

対象バージョン: **v0.11.0-beta**
配布物: Android APK / Windows zip (ポータブル・未署名) / Web (https://kurage.demo2.jp)

> ⚠️ クローズドベータ版です。まれに再ログインやデータ消去が必要になることがあります。
> Windows zip は未署名のため、初回起動時に「Windows によって PC が保護されました」と出たら **「詳細情報」→「実行」** で起動してください。

---

## ✨ 新機能

- **クリップボードの画像を貼り付けて添付できるように** (Web / Windows・macOS・Linux)
  投稿画面で `Ctrl/Cmd+V`、または添付メニューから、コピーした画像をそのまま添付できます。複数アカウントへのクロスポストにも対応。（モバイルは対象外）
- **下書きに「CW」と「投票」も保存されるように**
  投稿を中断して閉じたときの自動下書き・明示保存の下書きの両方で、これまで失われていた CW（閲覧注意）と投票（選択肢・複数選択・期限）が再開時に復元されます。

## 🛠 改善・修正

- **探索（検索）ページのアカウント切替を刷新**
  ドロップダウンから、アバター + 名前 + ▼ のコンパクトな表示に変更し、一覧はプロフィール画面と同じ下から出るシートに統一。名前が長いと「探索」タイトルが消えてしまう問題も解消しました。
- **マイプロフィールのアカウント一覧でアバターが横長楕円に潰れる問題を修正**（Web）
- **DM（ダイレクトメッセージ）の表示が復活してしまう不具合を修正**
  アカウントを全解除した直後や、素早く選択を切り替えたときに「未選択なのに DM が出る」競合状態を解消しました。

## 📝 既知の注意点

- Windows 版はポータブル zip です。解凍して `kurage.exe` を実行してください（インストール不要）。設定・アカウントは `%APPDATA%\Kurage` に保存されるため、フォルダを移動しても引き継がれます。
- Windows 版にはプッシュ通知（FCM）はありません（Android / Web のみ）。
- アルファ/ベータ期間中のため、不具合に気づいたらお手数ですが開発者までご連絡ください。
