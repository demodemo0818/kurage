# 🪼 Kurage v0.16.1-beta リリースノート

v0.16.0-beta で残っていた通知アイコンの問題と、投稿編集まわりの修正です。

- 配布: Android (Google Play クローズドテスト) / Windows ポータブル版 (zip) / Web https://kurage.demo2.jp
- これはベータ版です。仕様変更や、まれに再ログイン・データ消去が起こる場合があります。
- Windows 版は未署名のため、初回に「Windows によって PC が保護されました」と出たら「詳細情報」→「実行」で起動できます（同梱の「お読みください.txt」にも記載）。

## 🛠 改善・修正

- **プッシュ通知のアイコンが白い四角 (■) になる問題を修正しました**
  - v0.16.0-beta で通知は届くようになったものの、アイコンが ■ のままでした。Google Play 配布時のリソース分割が原因でクラゲアイコンを読み込めない端末があったため、確実に読み込める場所にもアイコンを配置しました。今回の更新後はクラゲのシルエットが表示されます。
  - 「設定 → 通知 → プッシュ通知の状態を確認」にテスト通知を追加しました。実行するとその場で通知が1件届き、アイコンが正しく出るかを確認できます。
- **投稿の編集中に「予約」を設定できてしまう問題を修正しました**
  - 公開済み投稿の編集を予約する機能は Mastodon 自体に存在せず、設定しても保存時に無視されて「保存できたのに何も変わらない」ように見えていました。編集中は「予約」ボタンを表示しないようにしました。
  - 投稿を予約付きで出し直したい場合は、投稿メニューの「削除して下書きに」→ 予約設定 → 投稿、をご利用ください。
