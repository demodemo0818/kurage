# 🪼 Kurage v0.8.2-alpha リリースノート

前回（v0.8.1-alpha）からの更新です。今回は不具合修正のみです。

- 配布: Android APK / Web 版（https://kurage.demo2.jp）
- これはクローズドアルファ版です。仕様変更や、まれに再ログイン・データ消去が起こる場合があります。

## 🛠 修正

- **ボスモード（偽装モード）から戻ったあと、一部の操作ができなくなる不具合を修正**（Web / デスクトップ）
  - F9 で偽装モードに入って戻ると、その後
    **カラムの ⋮ メニュー・投稿ペインのアカウント選択・画像のポップアップ表示**
    が開かなくなることがありました。
  - ページを再読み込み（F5）するまで直らない状態でしたが、再読み込み不要で正常に動くようになりました。
